About This Project

Hello! My name is David Varga, and I'm an aspiring web developer with a strong passion for fantasy storytelling, game design, and user interface development.
This project, Tales of Eldoria, is a fictional fantasy RPG website designed to showcase my front-end skills, including responsive design, layout structuring, CSS styling, and interactive components.

What I Used

  • HTML5, semantic structure
  • CSS3 with custom styling and media queries
  • Bootstrap 5 components
  • Responsive design techniques
  • JavaScript for creating the audio button and dynamic footer
  • Font Awesome for icons
